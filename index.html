Egyptian Mythology API

Welcome to the Egyptian Mythology API! Here you will find information about gods, creatures, symbols and much more..

Endpoint base: https://mariviana.dev/egyptian-mythology-mock-api/v1

Endpoints:

Gods

  • List: /gods/gods.json
  • Item: /gods/{id}.json
  • Example Item /gods/ra.json

Creatures

  • List: /creatures/creatures.json
  • Item Endpoint: /creatures/{id}.json
  • Example Item /creatures/ammit.json

Symbols

  • List: /symbols/symbols.json
  • Item Endpoint: /symbols/{id}.json
  • Example Item /symbols/ammit.json

Stories

  • List: /stories/stories.json
  • Item Endpoint: /stories/{id}.json
  • Example Item /stories/ammit.json

Places

  • List: /places/places.json
  • Item Endpoint: /places/{id}.json
  • Example Item /places/ammit.json
